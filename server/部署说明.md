# HwMonServer 部署指南

## 快速开始

### 1. 配置数据库连接

在运行服务端之前，需要编辑 `config.json` 文件来配置数据库连接信息：

```json
{
  "database": {
    "host": "192.168.20.17",    // MySQL服务器地址
    "port": 3306,                // MySQL端口
    "user": "HwMon",             // 数据库用户名
    "password": "kk7cy7SDWDMXC5XQ",  // 数据库密码
    "database": "hwmon",         // 数据库名称
    "charset": "utf8mb4"
  },
  "login": {
    "username": "xapi",          // Web登录用户名
    "password": "Ai78965"        // Web登录密码
  },
  "server": {
    "port": 5000,                // 服务端监听端口
    "host": "0.0.0.0"            // 监听地址
  },
  "collect": {
    "max_workers": 50,           // 最大并发采集数
    "timeout": 15,               // 单个客户端超时时间(秒)
    "retry_times": 0             // 失败重试次数
  }
}
```

### 2. 启动服务端

双击 `启动服务端.bat` 或运行：
```bash
HwMonServer.exe
```

### 3. 访问Web界面

打开浏览器访问：`http://localhost:5000`

首次访问需要登录：
- 用户名：xapi
- 密码：Ai78965

## 配置说明

### 数据库配置 (database)

| 参数 | 说明 | 默认值 |
|------|------|--------|
| host | MySQL服务器IP地址 | 192.168.20.17 |
| port | MySQL端口号 | 3306 |
| user | 数据库用户名 | HwMon |
| password | 数据库密码 | kk7cy7SDWDMXC5XQ |
| database | 数据库名称 | hwmon |
| charset | 字符集 | utf8mb4 |

### 登录配置 (login)

| 参数 | 说明 | 默认值 |
|------|------|--------|
| username | Web管理界面登录用户名 | xapi |
| password | Web管理界面登录密码 | Ai78965 |

### 服务器配置 (server)

| 参数 | 说明 | 默认值 |
|------|------|--------|
| port | 服务端监听端口 | 5000 |
| host | 监听地址（0.0.0.0表示所有网卡） | 0.0.0.0 |

### 采集配置 (collect)

| 参数 | 说明 | 默认值 |
|------|------|--------|
| max_workers | 最大并发采集线程数 | 50 |
| timeout | 单个客户端请求超时时间(秒) | 15 |
| retry_times | 失败重试次数 | 0 |

## 常见问题

### Q1: 如何修改数据库地址？

编辑 `config.json` 文件，修改 `database.host` 字段为你的MySQL服务器地址。

### Q2: 如何修改Web登录密码？

编辑 `config.json` 文件，修改 `login.password` 字段。

### Q3: 如何修改服务端端口？

编辑 `config.json` 文件，修改 `server.port` 字段，然后重启服务端。

### Q4: config.json 文件在哪里？

- **开发环境**：与 `app.py` 同目录
- **打包后**：与 `HwMonServer.exe` 同目录

### Q5: 修改配置后需要重启吗？

是的，修改 `config.json` 后需要重启服务端才能生效。

## 部署步骤

1. **准备MySQL数据库**
   - 确保MySQL服务器正在运行
   - 创建数据库用户并授权
   - 开放3306端口（如果需要远程访问）

2. **配置服务端**
   - 编辑 `config.json` 文件
   - 修改数据库连接信息
   - 修改登录凭证（建议）

3. **启动服务端**
   - 双击 `启动服务端.bat`
   - 或在命令行运行 `HwMonServer.exe`

4. **验证运行**
   - 访问 `http://localhost:5000`
   - 使用配置的账号密码登录
   - 确认可以正常访问

5. **配置防火墙**
   ```bash
   netsh advfirewall firewall add rule name="HwMonServer" dir=in action=allow protocol=TCP localport=5000
   ```

## 注意事项

⚠️ **安全提示**：
- 生产环境请修改默认登录密码
- 不要将 `config.json` 文件提交到版本控制系统
- 建议使用强密码保护数据库
- 定期备份数据库数据

📝 **配置文件位置**：
- 配置文件必须在 exe 文件同目录下
- 如果找不到配置文件，程序会使用内置的默认配置
- 首次运行时会自动检测配置文件是否存在
