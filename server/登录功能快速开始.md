## ❓ 常见问题

### Q1: 提示"用户名或密码错误"？

**检查**:
- 用户名是否是 `xapi`（全小写）
- 密码是否是 `Ai78965`（注意大小写）
- 是否有多余的空格

**解决**:
重新输入，确保完全正确。

### Q2: 登录后刷新页面又要登录？

**原因**: 
- 浏览器禁用了Cookie
- 使用了隐私/无痕模式

**解决**:
- 启用浏览器Cookie
- 不要使用无痕模式
- 清除浏览器缓存后重试

### Q3: 客户端还能上报数据吗？

**答案**: ✅ 可以！

客户端上报API (`/api/report`) 是公开的，不需要登录。
只有管理界面和API需要登录验证。

### Q4: 如何保持登录状态？

**说明**:
- 只要不关闭浏览器，登录状态会一直保持
- 关闭浏览器后需要重新登录
- 目前没有"记住我"功能

### Q5: 多人可以同时登录吗？

**答案**: 可以，但需要注意：
- 每个浏览器有独立的会话
- 同一浏览器多个标签页共享会话
- 一个标签页登出，其他也会失效

## 🔒 安全提示

### 生产环境部署建议

1. **修改默认密码**
   ```python
   # 在 app.py 中修改
   LOGIN_CONFIG = {
       'username': 'your_username',
       'password': 'your_strong_password'
   }
   ```

2. **启用HTTPS**
   - 使用SSL证书
   - 加密数据传输
   - 防止密码泄露

3. **限制访问IP**
   - 只允许内网访问
   - 或配置VPN

4. **定期更换密码**
   - 提高安全性
   - 防止密码泄露
